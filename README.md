# Dentioo Frontend

Frontend application for Dentioo dental practice management system built with Next.js 15, React, and TypeScript.

## Features

- Patient management
- Appointment scheduling
- Budget/quote system
- Financial tracking
- File management
- Authentication

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Shadcn/ui components

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
